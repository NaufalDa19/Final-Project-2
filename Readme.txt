Cara Menggunakan Program Preprocessing Dalam Suatu Data !!!!!!!!

-> Pastikan Anda memiliki aplikasi text editor pycharm ataupun visual studi code yang sudah di installkan ekstensi python terbaru

-> Download zip dari                                                          - Program Utama (main.py) 
   TT4404_Kelompok Apaya_1101204040_1101202216_1101204005_1101204116          - Program pengolah datanya yang mana di program ini berisi fungsi-fungsi (PengolahData.py)
   setelah didownload, pilih folder "Project Program Preprocessing Data"      - dataset1.csv (sebelum diolah)
   yang mana dalam satu folder itu berisi penjabaran data seperti disamping   - dataout.csv (data bersih yang sudah diolah)
                                                                              - Readme.txt

-> Setelah itu ekstrak file zipnya

-> Setelah di ekstrak jika programnya tidak mau berjalan berarti ada masalah di delimiternya,
   jika di dalam delimiter dari dataset1.csv belum benar bisa diganti delimiternya menjadi ";" pada excel, 
   dengan cara blok semua di kolom A yang mana kolom A ini terdiri dari Header NIM, 
   setelah itu pergi ke Data, setelah itu cari "Teks ke Kolom" lalu pilih delimited, dan pilih deliiternya yang menggunakan ";"

-> Setelah di ekstrak, buka aplikasi visual studio code lalu open folder di visual studio code

-> Kita masukan folder yang sudah kita ekstrak tadi ke visual studio code

-> Buka file dataset1.csv dalam aplikasi visual studio code nya

-> Pastikan dalam folder ini sudah termasuk dari data data yang ada diatas dan di dalam satu folder yang sama

-> Setelah itu klik run pada visual studio code untuk menjalankan programnya

-> Cek perbandingan data pada dataset1.csv dengan dataout.csv jika ada program yang dijalankan berhasil mengubah data dari dataset1.csv ke dataout.csv maka anda telah berhasil melakukan preprocessing dari datanya

-> Yang mana disini perubahan datanya adalah untuk menempatkan nim - nim yang tertukar di header lain ke tempat yang seharusnya
